SteamWatch 操作指南

一、启动程序
  如果使用安装器版：双击 SteamWatchSetup-win-x64.exe，安装完成后从桌面或开始菜单启动 SteamWatch。
  如果使用绿色版：解压后打开 SteamWatch-win-x64 文件夹，双击 SteamWatch.exe 启动。
  绿色版同目录下不要删除 .dll、.json、Assets、runtimes 等文件，它们是程序运行所需文件。

二、运行数据
  安装器版会安装到当前用户目录，不需要管理员权限。
  程序会在安装目录或绿色版目录下的 data\ 文件夹保存设置、限额规则和游玩记录。
  如果要迁移到另一台电脑，可以一并复制 data 文件夹。
  SteamWatch 运行时会按分钟记录游玩时间。
  未开启 SteamWatch 期间的游玩时间会在下次启动时根据 Steam 累计时长差值补记到当天。
  如果跨天游玩后才启动，Steam 本地缓存无法提供精确日期，补记不会拆分到历史日期。

三、通知提醒
  SteamWatch 支持系统通知、程序内弹窗和声音提醒。
  如果全屏游戏时收不到系统通知，通常是 Windows 自动开启了“勿扰模式”。

四、关闭 Windows 勿扰模式
  1. 按 Win + I 打开 Windows 设置。
  2. 点击左侧“系统”，再点击右侧“通知”。
  3. 确保顶部“勿扰模式”处于关闭状态。
  4. 展开“自动打开勿扰模式”。
  5. 取消勾选“玩游戏时”。
  6. 取消勾选“在全屏模式下使用应用时”。

五、托盘和退出
  程序运行后会显示托盘图标。
  可以在设置中选择关闭窗口时“最小化到托盘”或“退出程序”。
  托盘右键菜单可以打开窗口、暂停/恢复监控、进入设置或退出程序。
  如果在设置中开启“修改规则和退出时需要密码”，退出程序前需要输入认证密码。

六、游戏限额
  在“游戏监控”界面选择游戏后，可以为单个游戏设置每日或每周限额。
  也可以设置全局限额，作用于所有被监控的 Steam 游戏。
  “仅提醒”会发送提醒；“强制退出”会在倒计时后尝试关闭对应游戏进程。
  开启认证保护后，保存、更新或删除限额规则都需要输入密码。

七、认证保护
  可以在“设置”中开启“修改规则和退出时需要密码”，并设置认证密码。
  密码不会明文保存，程序只保存带盐哈希。
  该功能用于减少从 SteamWatch 应用内绕过规则的可能。
  它不能阻止通过任务管理器结束进程、删除本地 data 文件夹或卸载程序这类系统级操作。
